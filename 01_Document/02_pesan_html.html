1. Kategori 4xx (Kesalahan Klien) Terjadi saat browser salah mengirimkan permintaan atau halaman tidak dapat dijangkau. [] • 400 Bad Request: Server tidak dapat memproses permintaan karena format URL atau sintaks salah. • 401 Unauthorized: Pengguna belum login atau memerlukan autentikasi. • 403 Forbidden: Pengguna tidak memiliki izin untuk mengakses halaman tersebut. • 404 Not Found: Halaman atau URL tidak ditemukan. Biasanya karena salah ketik, dihapus, atau dipindahkan. [1, 2] 2. Kategori 5xx (Kesalahan Server) Terjadi saat server gagal menjalankan permintaan yang sebenarnya valid dari pengguna. [1] • 500 Internal Server Error: Kesalahan umum yang menandakan adanya masalah atau bug di server. • 502 Bad Gateway: Server menerima respons buruk dari server lain. • 503 Service Unavailable: Server sedang sibuk, overload, atau dalam masa pemeliharaan. [1, 2] 3. Kategori Lainnya (Informasi & Sukses) • 200 OK: Permintaan berhasil diproses tanpa ada masalah. • 301 Moved Permanently: Halaman telah dipindahkan secara permanen ke URL baru.
